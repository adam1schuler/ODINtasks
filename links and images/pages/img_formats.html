An exapmle of the 4 file types, can be downloaded from here: mochi images

The four images are:

  1. JPG images are good for photos.
    • designed for handling large color palettes without exorbitantly increasing file size.
    • great for photos and images with lots of gradients in them
    • don’t allow for transparent pixels,
  2. GIFs are good for animations.
    • the go-to option for simple animations,
    • never use them for photos
    • better to use PNG images if you don’t need animation.
  3. PNGs are good for diagrams and icons.
    • great for anything that’s not a photo or animated.
    • would generally be bigger than an equivalent JPG file.
    • don’t have color palette limitations.
    • they do deal with opacity just fine
    • excellent fit for icons, technical diagrams, logos
  4. SVGs are amazing. Use them wherever you can.
    • SVG is a vector-based graphics format
    • can scale up or down to any dimension without loss of quality
    • all the same use cases as PNGs
